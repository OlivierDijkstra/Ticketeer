# TICKETGATE DASHBOARD

- I need to find a better name\*

This project is setup with bun:
https://bun.sh/

macos: `curl -fsSL https://bun.sh/install | bash`
windows: `powershell -c "irm bun.sh/install.ps1 | iex"`

## Setup

1. Install bun
2. Run `bun install`
3. Run `bun dev`

## Testing

Run `bun test`
